# E-commerce TCC

E-commerce project developed using Spring Boot.<br>

Login: admin@gmail.com <br>

## Used Technologies:

* Back-end: Spring (Boot, Data, Security), JPA / Hibernate, PostgreSQL, JUnit, Mockito
* Security: JWT
* REST API
* Server Build: Maven
